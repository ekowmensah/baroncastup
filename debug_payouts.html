Clear Browser Cache Instructions

Chrome/Edge: Ctrl+Shift+R (or Cmd+Shift+R on Mac)

Firefox: Ctrl+F5 (or Cmd+Shift+R on Mac)

Safari: Cmd+Option+R

Or open Developer Tools (F12), right-click refresh button, and select "Empty Cache and Hard Reload"

// Test if they work (if they exist)

What to Check

  • Open Developer Tools (F12)
  • Go to Console tab
  • Look for any red error messages
  • Click Network tab, refresh page, and check if correct files are loading
  • Check if dashboard.php or history.php are being loaded (not dashboard_backupp.php)

Common Issues

  • Browser Cache: Old JavaScript cached - clear cache and hard refresh
  • Wrong File Loading: Check Network tab for which file is actually loaded
  • JavaScript Errors: Check Console for syntax errors
